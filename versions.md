# Версии

## 1.7.4 (2026-06-15)
- В группу `Зал` добавлены должности `администратор`, `администратор-кассир`, `кассир`, `оператор`.
- Правило применяется ко всем ресторанам, включая `Коминтерна 166 CALL CENTRE`.

## 1.7.3 (2026-06-11)
- Удален блок синхронизации с Google Sheets и инструкция Apps Script.
- Интерфейс оставлен под загрузку файлов, расчет и экспорт в CSV/Excel.

## 1.7.2 (2026-06-08)
- Добавлена поддержка нового списка сотрудников с колонкой `Подразделение`.
- Автоопределение файлов теперь распознает списки сотрудников с `Название подразделения` и `Подразделение`.

## 1.7.1 (2026-05-22)
- Блок инструкции перенесен выше блока Google Sheets.
- В инструкции добавлены кликабельные ссылки на страницы Saby.
- Добавлен отдельный блок `Следующий шаг для вас` с 2 шагами настройки интеграции.

## 1.7.0 (2026-05-22)
- Добавлена общая загрузка файлов с автоопределением типа (проходная/сотрудники).
- Добавлен блок мини-инструкции по выгрузке файлов из Saby.
- Добавлена отправка рассчитанных данных в Google Sheets через webhook (Apps Script URL).
- Добавлен файл `google-apps-script.md` с готовым шаблоном webhook.

## 1.6.2 (2026-05-05)
- Добавлена пометка конфликтных ФИО в строках дня/ресторана.
- В блоке `Показать` конфликтные сотрудники помечаются бейджем `конфликт ФИО`.

## 1.6.1 (2026-05-05)
- Добавлена совместимость новой выгрузки проходной с колонкой `ФИО` (вместо `Фамилия/Имя/Отчество`).
- Сохранена поддержка старого и нового форматов файла проходной.

## 1.6.0 (2026-03-17)
- Добавлена отдельная группа `Бар` (бармен, барменеджер, барбэк).
- Группа `Бар` добавлена в фильтр, таблицу, пофамильный разворот и выгрузки CSV/Excel.

## 1.5.2 (2026-03-15)
- Добавлен cache-busting для `script.js` (`script.js?v=1.5.2`), чтобы сайт не оставался на старой версии из кэша браузера.

## 1.5.1 (2026-03-15)
- Добавлен workflow `.github/workflows/deploy-pages.yml` для автопубликации GitHub Pages при `push` в `main`.

## 1.5.0 (2026-03-15)
- Изменена граница операционного дня: сутки считаются с 04:00 до 03:59.
- Время после полуночи до 04:00 относится к предыдущему дню.
- Расчет смен учитывает ночные выходы следующего календарного дня в рамках одного операционного дня.

## 1.4.1 (2026-03-15)
- Версия вынесена в заметный бейдж в шапке страницы.
- Обновлен интерфейс для явной индикации текущей сборки.

## 1.4.0 (2026-03-15)
- Добавлен разворот пофамильно в таблице результатов.
- Для каждой строки (дата + ресторан) доступен список сотрудников по группам: Кухня, Зал, Доставка.
- В списке по каждому сотруднику показывается вклад в смены (`1` или `0.5`).

## 1.3.0 (2026-03-15)
- Добавлена загрузка файла списка сотрудников (`ФИО` + `Название подразделения`).
- Ресторан в аналитике теперь может проставляться по списку сотрудников (маппинг по ФИО).
- Добавлен экспорт итогов в Excel-матрицу: строки — рестораны, столбцы — даты.
- Для Excel-экспорта формируются листы по выбранным группам и лист `Итого`.

## 1.2.0 (2026-03-12)
- Добавлен расчет смен по времени пребывания: больше 7 часов = 1 смена, до 7 часов = 0.5.

## 1.1.0 (2026-03-12)
- Добавлены фильтры по ресторанам, датам и группам должностей.
- Добавлен экспорт результата в CSV.

## 1.0.0 (2026-03-12)
- Базовая версия интерфейса с загрузкой файла проходной и расчетом по группам должностей.
