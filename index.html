Количество работающего персонала в день

Версия:

Источник данных: проходная. Смена: больше 7 часов = 1, до 7 часов = 0.5. Сутки считаются с 04:00 до 03:59.

Можно выбрать сразу оба файла. Сайт сам определит: проходная или список сотрудников.

Загрузите файл проходной.

Где взять файлы в Saby

  1. Файл проходной: открыть Графики работ → Пришел/ушел, выбрать период и фильтр Проходная, нажать Выгрузить.
  2. Список сотрудников: открыть Сотрудники, в фильтре выбрать Работающие за нужный месяц.
  3. Перед выгрузкой списка убрать подразделения, которые дают дубли (Сдача отчетности и сторонние организации).
  4. Далее Меню → Выгрузить → Список сотрудников в Excel (набор данных Основная информация).

Синхронизация с Google Sheets

Нужен опубликованный Apps Script webhook с правом записи в таблицу.

Следующий шаг для вас

  1. Создать Apps Script по инструкции из google-apps-script.md и получить Web app URL.
  2. Вставить Web app URL и Spreadsheet ID в блоке выше, затем нажать Отправить в Google Sheets.
